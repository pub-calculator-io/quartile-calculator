Use our free Quartile Calculator to easily find Q1, Q2 (median), Q3, interquartile range (IQR), minimum, maximum, and dataset range in seconds.

[https://www.calculator.io/quartile-calculator/](https://www.calculator.io/quartile-calculator/)

== Usage ==

Add the Quartile Calculator shortcode to your page, post or sidebar:

`[calcio_quartile_calculator]`

Add the following code to your template where you would like the Quartile Calculator to appear:

`<?php display_calcio_quartile_calculator(); ?>`

== Installation ==

1. Upload the Quartile Calculator /calcio_quartile_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Quartile Calculator plugin through the "Plugins" menu in WordPress.
